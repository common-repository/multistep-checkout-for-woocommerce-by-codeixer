== Description ==

With this plugin the Buyers of your website will get a new step by step User Interface for checkout page.Not only that,This plugin will improve your conversion rate.
<br>
= Features: =
<li> Two different Themes </li>
<li> Responsive layout </li>
<li>Left & Right arrow key Navigation  on Keyboard  </li>
<li> Setting panel </li>

= <a href="https://demo.codeixer.com/twist/">Live Preview</a> =

== Installation ==


1. Upload \`multistep-checkout\` folder to the \`/wp-content/plugins/\` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
=How do I install this plugin?=
You can install as others regular wordpress plugin. No different way. Please see on installation tab.
